4_json
======

Script for pretty print JSON.

Usage
-----

```
~$ python3 pprint_json.py 
Enter the path to the JSON file: example.json
[
    {
        "Id": "79742784",
        "Cells": {
            "global_id": 14371450
        },
        "Number": 1
    }
]
```

Reruirements
------------

- Python >= 3.4
